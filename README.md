# Adidas Data Insights & Sales Performance Dashboard

## Overview
A comprehensive, multi-page business intelligence dashboard for Adidas sales analytics, providing deep insights into sales performance across products, regions, sales methods, and time periods. This interactive dashboard enables strategic decision-making through detailed sales, profit, and retail analytics.

## Dashboard Navigation

The dashboard consists of five main pages:
1. **Home/Landing Page** - Welcome screen with call-to-action
2. **Overview Dashboard** - Key metrics, monthly trends, and channel analysis
3. **Product Performance** - Product-level sales and profitability analysis
4. **Regional Analysis** - Geographic performance and regional insights
5. **Profit Analysis** - Detailed profitability breakdown by multiple dimensions

### Side Navigation Menu (All Pages)
- **Adidas Logo** - Brand identity
- **Home Icon** - Return to landing page
- **Analytics Icon** - Sales performance dashboard
- **Map Icon** - Regional analysis (highlighted in screenshots)
- **Money Icon** - Profit analysis
- **Exit Icon** - Exit or logout

## Technical Specifications

### Dashboard Platform
- **Tool**: Power BI or similar business intelligence platform
- **Pages**: 5 interactive pages
- **Interactivity**: Cross-filtering across all visualizations
- **Responsive Design**: Optimized for desktop viewing
- **Navigation**: Sidebar menu with icon-based navigation

### Data Architecture
- **Time Granularity**: Monthly, yearly
- **Geographic Levels**: Regional (5 regions)
- **Product Hierarchy**: Product → Category (6 products)
- **Channel Types**: In-store, Outlet, Online
- **Retailer Data**: Individual retailer tracking (9,648 total)

### Metrics & Calculations
- **Total Sales**: Sum of all transactions ($900M)
- **Total Quantity**: Sum of units sold (2M)
- **Total Profit**: Revenue minus costs ($332M)
- **AVG Profit Margin**: Profit ÷ Sales (42%)
- **Derived Metrics**: Revenue per unit, profit per region, etc.

### Filter Functionality
- **Region Filter**: Single-select dropdown
- **Month Filter**: Single-select dropdown
- **Cross-page Filtering**: Filters persist across pages
- **Default State**: All filters set to "All"
